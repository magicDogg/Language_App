# Language_App

# Run local server:

1. cd (ROOT)/mysite
2. PYTHON manage.py runserver
3. Open in browser, and navigate to: http://127.0.0.1:8000/first_module and/or http://127.0.0.1:8000/second_module 

# To create a new app/module:
1. cd (ROOT)/mysite
2. PYTHON manage.py startapp <app_name>
